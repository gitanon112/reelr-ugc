Secure Flow

How it works

  1. Authenticate with TikTok (OAuth + PKCE)
  2. Choose media (upload or AI generated)
  3. Set caption, cover timestamp, privacy
  4. Draft upload or direct post via API
  5. Poll status & confirm publication
POST /v2/post/publish/video/init\nPUT upload_url (chunked)\nGET /v2/post/publish/status/fetch\n→ state: PROCESSING → PUBLISHED

Security & Data Handling

We store only open_id, basic profile (display name, avatar URL), posting metadata (caption, chosen privacy), and ephemeral upload references. Media assets auto‑purge within 24h of successful posting or draft abandonment unless intentionally saved. Revocation triggers token wipe.
