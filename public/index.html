Unified TikTok Content Workflow

Reelr helps creators and brands generate, curate, and publish high‑quality short videos. Connect your TikTok account via OAuth (Login Kit) and choose either direct posting (video.publish) or draft upload (video.upload) for final in‑app editing.

Key Capabilities

  • Secure TikTok Login (Login Kit, user.info.basic)
  • Draft Upload & Direct Post (video.upload / video.publish scopes as approved)
  • Automated AI media generation (disclosed as AIGC when applicable)
  • Scheduling & metadata templating
  • Compliance: explicit privacy & terms, user consent, revocable access

Why TikTok App Review Should Approve

  1. Clear User Value: Streamlines legitimate creator workflow—no scraping or unsupported automation.
  2. Scope Minimization: Only requests required scopes (user.info.basic plus content posting).
  3. Transparent AIGC: We label AI‑generated media and respect platform policies.
  4. Revocation Ready: Users can disconnect at any time; we honor token revocation.

Security & Privacy

Access tokens are stored server‑side, encrypted at rest. We never request user passwords. Users may revoke authorization inside TikTok or via our support channel.
